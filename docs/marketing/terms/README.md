# Marketing Glossary

`marketing` · `glossary` · **Level:** Beginner

> [!NOTE]
> Index of plain-English definitions for the marketing, pricing, and app-store terms used across the marketing strategy docs. Start here if a term is unfamiliar.

## What it is

A glossary, grouped by topic. Each file explains its terms in everyday language first, then gives the precise meaning, a real number, and why it matters. Read the relevant file before the strategy docs, which assume these terms.

_The four term groups:_

| File | Covers | Read it when... |
| :--- | :--- | :--- |
| [pricing-models.md](pricing-models.md) | Freemium, hard paywall, free trial, bundle | You see "paywall," "freemium," or "bundle" |
| [metrics.md](metrics.md) | Conversion, churn, ARPU, LTV, MRR | You see "conversion rate," "churn," or "LTV" |
| [influencers.md](influencers.md) | Nano/micro/macro, barter, revenue share, promo code, engagement | You see "nano-influencer," "barter," or "promo code" |
| [app-store.md](app-store.md) | Commission, Small Business Program, SKU, offer code | You see "commission," "offer code," or "SKU" |

---

## Why / when to use it

The strategy docs — [subscription-bundle-and-influencer-attribution.md](../subscription-bundle-and-influencer-attribution.md) (the decisions) and [creator-outreach-and-barter-ops.md](../creator-outreach-and-barter-ops.md) (the day-to-day outreach playbook) — make decisions _using_ these terms (e.g. "hard paywall converts 5× better than freemium"). If a sentence there is unclear, the underlying term is defined in one of the four files above.

---

## Key points / gotchas

- **Read the term first, the strategy second.** The strategy doc assumes the vocabulary.
- **Definitions are grouped, not one-per-file** — related terms are easier to understand together.
- **Every definition includes a real number** so the term is concrete, not abstract.

---

## Common mistakes

- **Reading the strategy doc cold** and getting stuck on a term that is defined here.
- **Treating these as final strategy** — they only define vocabulary; decisions live in the strategy doc.
